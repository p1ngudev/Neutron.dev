## Installation Guide

1. Install the latest version of **[Node.JS](https://nodejs.org/en/download/current/)**.
2. Download **[Neutron++](https://github.com/p1ngudev/Neutron.dev)**.
3. Extract the zip.
4. Open `install.bat` and wait for it to install. (Only required on the first run!)
5. Open `run.bat`, it should say `NeutronFN is listening on port 4432!` (DON'T CLOSE THIS WHILE USING NEUTRON!)
6. Launch the game **without a secret**!!!
7. On the login screen, put in `{YOURUSERNAME}@.` for the email and the password doesn't matter.
8. If everything worked correctly, you should be put into the lobby! Click `PLAY` if you wanna go inside of a game!
